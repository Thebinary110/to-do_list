To-Do List Project
This project is a simple to-do list application built using HTML, CSS, and JavaScript. It allows users to add tasks, mark them as completed, and delete tasks from the list.

Installation
To clone this repository to your local machine and view/edit the code in your code editor, follow these steps:

Open your terminal or command prompt.

Navigate to the directory where you want to clone the repository.

Use the following command to clone the repository:

bash
Copy code
git clone https://github.com/Thebinary110/to-do_list.git
Once the repository is cloned, navigate into the project directory:

bash
Copy code
cd to-do_list
Usage
After cloning the repository and navigating into the project directory, you can open the index.html file in your preferred web browser to use the to-do list application.

To-Do List Functionality
Add Task: Enter a task in the input field and press the "Add Task" button to add it to the list.
Mark Task as Completed: Click on the checkbox next to a task to mark it as completed. Completed tasks will have a line-through style.
Delete Task: Click on the "Delete" button next to a task to remove it from the list.
You can customize the appearance or functionality of the to-do list by modifying the HTML, CSS, and JavaScript files in your code editor.
